<br />

Syntex is very simple.<br />
Just putting _abstract_ keyword before _class_ as a prefix will make an abstract class.<br />

Abstract class Can't have its own instance just like the interface. <br />
But declaring abstract class type and upcasting just like the interface.

And Abstract class can have normal methods and __abstract methods__. <br />

To declare abstract method, you have to put the _abstract_ keyword again before the _return type_ of that method.<br />
This is similar to _virtual_ keyword because to implement this function derived class has to have _override_ keyword to override.

What is the function of abstract method?<br />
It is exactly the same with __interface__, just forcing derived class to implement some particular methods.

At the line 34, we are creating a dog instance that can access to Walk, Defecation method.<br />
However, there is another method, Born(). To be able to invoke that method, upcasting is needed.<br />

As You can see like 41 and 42, mammal instance is also invoking Walk and Defecation because we overrided them.

We have learn the fact by the code above that abstract class can have non-abstract methods.<br />
Unlike interfaces, derived class has no obligation to implement that non-abstract class.

<hr />

### Sum up

> __Differences:__<br />
> - Unlike interfaces, default access specifier is private.<br />
> - Abstract class can have non-abstract methods and upcasting is needed when you try to invoke them.
> - Interface's methods are public, Abstract methods are also public but able to be protected.
> - The interface is multi-inheritable, but abstract class is not.

> __Similarities:__<br />
> - Both cannot have its own instances. Only upcasting is available.
> - Abstract class can have abstract method and its function is just the same with interface's methods.<br />

Interfaces and abstract classes are very similar to each other, but slightly different.<br />
You can choose which one is better at some situation based on your experience on OOP programming and knowledge.
