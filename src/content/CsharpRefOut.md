## ref

_ref_ keyword is very useful keyword when it comes to methods.

In most cases, arguments of methods is value type.<br />
So whenever you try to change the value of parameters inside a function or method, the change won't affect outside of the method.

Let's look at the code to help us understand:

As you can see, when we pass a variable which is value type, then it will stay still even if you assign any value to that parameter.<br />
When you pass arguments to a method, the parameter of a method will copy the value of those arguments.<br />
So the parameters int a and b is totally different variable than the integers declared in the _main_ function.<br />

You may wonder what can we do when we want to change the parameter inside the function??<br />
